# Examples

Examples are grouped by usage domain.

## widgets

- widgets/progressbar_dual.py: Two progress bars filling at different speeds.
- widgets/choice_palette.py: Choice widget with color palette selection.
- widgets/canvas_bounce.py: Animated canvas with a bouncing ball.

## multimodal

- multimodal/focus_routing.py: Keyboard + mouse focus routing between Choice and Input widgets.

## showcase

- showcase/sequential_journey.py: End-to-end multi-stage demo (progressbar -> choice -> input -> canvas).

Run an example from the repository root:

python examples/widgets/canvas_bounce.py
